@startuml
title Пошаговое взаимодействие в системе побега пингвинов "Мадагаскар"

actor "Шкипер" as S
participant "Ковальски" as K
participant "Рико" as R
actor "Младший" as P
participant "Охрана" as G
participant "Отмычка" as LP
participant "Транквилизатор" as T

' 1. Разработка плана
S -> K : рассчитать маршрут()
K --[#blue]> S : "Маршрут готов"

' 2. Выдача инструментов
S -> R : выдать инструмент("взлом")
R -> LP ** : создать()
note right of LP: инструмент для взлома

' 3. Отвлечение охраны
S -> P : отвлечь охрану()
P -> G : отвлекающий манёвр()
G --[#green]> P : "Отошёл на шум"

' 4. Взлом замка
S -> K : взломать замок()
K -> LP : использовать()
note over K #lightgreen: Замок ВЗЛОМАН

' 5. Усыпление охраны
S -> R : усыпить охрану()
R -> T ** : создать()
note right of T: транквилизатор
R -> G : выстрелить()
destroy G
note over G #FFAAAA: Охрана УСЫПЛЕНА

' 6. Захват судна
S -> S : захватить судно()
note over S #lightgreen: Судно ЗАХВАЧЕНО

@enduml
